Quick User Guide:
Run the program and insert inputs:
1. function inputs options - "add_file", "add_dir", "delete", "show_system_file"
2. only folders can have no parent folder (files must have parent folder)



--------------------- הנחות -------------------
1. מחיקת תיקייה מוחקת גם את כל הקבצים והתיקיות שבתוכה
2. יכולות להיות מספר תיקיות "בסיס" כלומר אין שורש יחיד (אחרת הייתי מממש מבנה של עץ Trie)
3. לא ניתן להוסיף לתיקיה את שם תיקיית האם שלה ולא ניתן לשמור מבנה כזה (בהתאם לאטריביוטים שמוגדרים לתיקיה)
4. היוזר מכניס קלטים תקינים (לא התמודדתי כמעט עם מקרי קצה)
5. לא מקבלים נתיב מלא (אלא רק שם) אחרת מבנה של TRIE היה מהיר בהרבה לביצוע הפעולות כמעט ב O(1)
